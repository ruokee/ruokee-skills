# 中文特有的 AI 写作模式

这份 reference 只补充中文特有模式。先应用 `phrases.md` 和 `structures.md` 中的通用原则，再检查下面的问题。命中词语不代表文本由 AI 生成，也不等于必须删除；结合作者语气、受众和上下文判断。

## 空泛时代背景

模型常用宏大背景代替具体上下文：

- “在当今快速发展的时代”
- “随着科技的不断进步”
- “在数字化浪潮席卷全球的背景下”
- “在这个充满机遇与挑战的时代”
- “伴随着……的深入发展”

删掉时代背景，直接写与主题有关的变化、时间和影响。

> 修改前：随着人工智能技术的飞速发展，代码审查正迎来前所未有的变革。
>
> 修改后：团队接入代码审查模型后，平均等待时间从 20 分钟降到 6 分钟，但误报率仍有 14%。

## 中文式起手和收尾

### 起手

- “值得一提的是”
- “需要注意的是”
- “不可否认的是”
- “毋庸置疑”
- “众所周知”
- “不难发现”
- “从某种意义上来说”

这些表达常替读者判断重要性或确定性。直接写事实；需要强调时写清风险、影响或证据。

### 收尾

- “总而言之”
- “综上所述”
- “由此可见”
- “让我们拭目以待”
- “让我们共同期待”
- “开启……新篇章”
- “共创更加美好的未来”

删除没有新增信息的总结。结尾应落在具体结论、限制、决定或下一步。

## 伪共情和疗愈腔

模型在日记、建议和情绪话题中容易套用未经请求的安慰：

- “请记住，你并不孤单”
- “这不是你的错”
- “你已经做得很好了”
- “允许一切发生”
- “给自己一点时间和空间”
- “愿你在……中找到属于自己的答案”
- “真正的成长，是学会……”
- “你要相信，一切都是最好的安排”

不要默认安慰、升华或替用户定义感受。回应对方已经表达的具体情绪和处境；对方只要求整理文字时，不添加心理判断。

## 四字词和对称句堆叠

连续四字词会形成公文或宣传稿腔调：

- “全面深入、系统高效、持续推进”
- “提质增效、降本增效、稳中有进”
- “同频共振、协同联动、凝心聚力”
- “可感可知、走深走实、见行见效”
- “有高度、有温度、有力度”

保留组织内部有固定含义的术语。其余改成具体动作、对象和结果，不要用另一组四字词替换。

## 公文、商业和产品黑话

检查这些词是否掩盖了行动者和实际动作：

| 表达 | 需要追问 |
| --- | --- |
| 赋能、助力、驱动 | 谁给谁提供了什么能力？ |
| 抓手、闭环、链路 | 具体流程和责任人是什么？ |
| 拉通、对齐、协同 | 哪些人确认了哪些信息？ |
| 沉淀、复用、反哺 | 保存了什么，谁会怎样使用？ |
| 打造、构建、布局 | 实际创建或修改了什么？ |
| 颗粒度、底层逻辑、方法论 | 指的是哪项指标、原因或方法？ |
| 全方位、多维度、系统性 | 范围和维度分别是什么？ |
| 组合拳、全链路、生态位 | 包含哪些措施、环节或角色？ |

这些词在特定团队中可能是有效术语。受众理解且含义明确时保留；不能落到具体对象时改写。

> 修改前：拉通上下游资源，形成交付闭环，持续赋能业务增长。
>
> 修改后：产品和销售共用一份客户清单，每周核对签约与交付状态。

## 互联网黑话和热词拼贴

模型会把流行词当作现成态度，尤其是在评论、社交媒体文案和生活方式内容中：

- 赛道、破圈、出圈、降维打击、天花板
- 松弛感、氛围感、情绪价值、配得感、主体性
- 内耗、能量场、课题分离、长期主义
- 上头、拿捏、封神、赢麻了、狠狠、绝绝子
- “直接一个……” “谁懂……” “狠狠共情了” “把……具象化了”

不要为了显得口语化而添加热词。原作者有意使用、符合社群语境或作为讨论对象时保留；否则写出具体感受、行为或判断。

## 空泛动词和名词化

中文 AI 文本常用“进行、实现、完成、推动、促进”把简单动作拉长：

- “对日志进行分析” → “分析日志”
- “实现效率的提升” → 写明时间、成本或步骤的变化
- “推动问题得到解决” → 写明谁解决了什么问题
- “完成对方案的优化” → 写明改了哪部分

不要机械删除。它们承担真实语义或符合正式文体时可以保留。

## 中文语法例外

上游规则以英语为中心，处理中文时不要机械套用：

- 中文可以省略上下文明确的主语。不要给每句强加“我们”“团队”或“用户”。
- 被字句、无主句和“文件已归档”这类表达可以自然且准确。行动者影响责任判断时再补充。
- 中文没有与英语 `-ly` 对应的副词删除规则。“仅、最多、当前、可能”等词会改变事实边界，必须保留。
- 中文破折号有正常语法和修辞用途。只清理为制造戏剧停顿而滥用的破折号。
- “不是 X，而是 Y”、排比和三项列表表达真实对比或结构时可以保留。删除的是模板，不是句式本身。

## 快速检查

- 是否用宏大时代背景代替了具体上下文？
- 是否替读者宣布内容“重要、显然、不可否认”？
- 是否自动添加了安慰、升华或祝愿？
- 四字词、对称句和热词是否密集出现？
- 黑话能否还原成具体的人、动作、对象或结果？
- 是否为了遵守英文规则破坏了自然中文或事实边界？
