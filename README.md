# ruokee-skills

Ruokee's personal Skill workspace repository.

This repository manages Skill workspaces, installable Skill bodies, third-party Skill references, and future installer tooling.

Current status:

- Repository initialized.
- No Skills have been added yet.

Planned top-level layout:

```text
skills/
├── public/
├── experimental/
└── third-party/
```

Directory meaning:

- `public/`: self-authored Skills that are stable enough to publish.
- `experimental/`: self-authored or adapted Skills still under development.
- `third-party/`: externally sourced Skills, usually kept as minimal workspaces with source and selection notes in `README.md`.
